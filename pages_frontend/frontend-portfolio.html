Overview

This personal website has been crafted to showcase my frontend development skills. Built with raw HTML, CSS, and JavaScript, it serves as a portfolio for my academic, personal, and professional projects. The design was created using Figma, and the website is hosted on GitHub Pages, making it accessible to a wide audience.

Project Goals

To provide a comprehensive overview of my frontend development projects and skills, demonstrating my ability to create responsive and visually appealing web applications.
